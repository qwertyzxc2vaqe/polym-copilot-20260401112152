AUTOMATIC RESUME MECHANISM
═══════════════════════════════════════════════════════════════════════════════

Iteration N:
  └─ is_paused = True, pause_until = T+10s
     (Loop continues running)
     └─ Check: now >= pause_until? NO
        └─ Sleep 1s, continue

Iteration N+1 (1s later):
  └─ is_paused = True, pause_until = T+10s
     └─ Check: now >= pause_until? NO
        └─ Sleep 1s, continue

...

Iteration N+10 (10s later):
  └─ is_paused = True, pause_until = T+10s
     └─ Check: now >= pause_until? YES! ✓
        └─ Set is_paused = False
           └─ Set pause_until = None
              └─ Log: "Pause expired, resuming"
                 └─ Continue to normal flow
                    └─ Scan for next 5-min window!
