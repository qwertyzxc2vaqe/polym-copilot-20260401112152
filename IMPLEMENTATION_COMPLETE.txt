WORKFLOW
===============================================================================

When 5-minute market expires at T=0:

1. DETECTION
   Trading loop iteration detects: active_market.seconds_to_expiry <= 0
   
2. STEP 1 - UNSUBSCRIBE
   Calls: await self._sniper.unsubscribe([yes_token_id, no_token_id])
   Effect: WebSocket closed, order book freed, bandwidth freed
   
3. STEP 2 - PAUSE
   Calls: self.pause_asset(asset, duration_seconds=10.0, reason="post-sale-cooldown")
   Effect: Scanning paused, is_paused=True, pause_until=now+10s
   
4. SKIP ITERATION
   Breaks early from loop iteration to skip remaining processing
   
5. AUTOMATIC RESUME (next iteration)
   Main loop checks: if pause_until has expired
   Effect: is_paused=False, pause_until=None, resume scanning
   
6. READY FOR NEXT WINDOW
   Continues scanning for next 5-minute market
